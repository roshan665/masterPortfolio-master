# Roshan Sahani Portfolio

This repository contains my personal portfolio website built with React.

## About

The portfolio includes:
- Home and introduction
- Skills and tools
- Education and experience
- Projects
- Contact section
- Resume viewer

## Tech Stack

- React
- React Router
- Base Web
- Styled Components
- React Reveal

## Run Locally

1. Install dependencies:
   npm install
2. Start development server:
   npm start
3. Create production build:
   npm run build
4. Run tests:
   npm test -- --watchAll=false

## Configuration

Most personal content is managed in:
- src/portfolio.js

## Deployment

You can deploy this project to GitHub Pages, Netlify, or any static hosting service.
